# @unocss/extractor-pug

Pug extractor for UnoCSS.

## Install

```bash
npm i -D @unocss/extractor-pug
```

```ts
import extractorPug from '@unocss/extractor-pug'

UnoCSS({
  extractors: [
    extractorPug(),
  ],
})
```
